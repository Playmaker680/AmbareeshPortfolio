AquaGuard AI Project

Date: Jan 2024 | Course: CSE

Oceans are getting polluted more by the day and the current solution isn't to the mark of solving this issue. This problem directly effects the aquatics and aquatic plants. My fellow classmates and I, together worked on a project called AquaGuardAI. AquaGuardAI is a AI based sensor which focuses on tackling sewage discharge and poor water pollution in our oceans. The AI model sensor aims to detect sewage discharge and a negative change in water quality in any given water body. The data will then be transmitted to a live-feed website which has a alarm system showing live sensor activity, location and possible detection. My role in this project was a Web Developer and Co-Project Manager. Specifically, I was responsible for leading the team with meetings, deadlines, coordinating with other members while ensuring that the project milestones were met. I also developed on the beta website which contains all the project information (Home, AI models, About Us, Sponsors) and the frontend of the live-feed page using HTML, CSS and Java Script. This project enhanced my technical skills, where I developed a seamless user friendly high designed website. It also taught me the importance of teamwork and effective communication amongst other workers.

TapFast Game

Date: Sep 2023 | Course: CSE

Boring rides, lack of interest in modern games, or just feeling minimalistic? TapFast is a multiplayer game where players compete to tap a random moving target the most amount of times in a given duration. The game aims a minimal yet fun way to play with your friends or practice on your own. This game's main objective besides fun competition is to improve reaction time and hand-eye coordination. As the Software Developer for this project, I developed the game mechanics, emphasizing the user experience in a intuitive UI/UX design. The game goes along a minimalistic design inducing competition as the main user objective. During development, I conducted various testing sessions to gather and document feedback, which I used to further improve the game's design and functionality. This project allowed me to further master MIT App Inventor programing which was foundational to my future projects.
